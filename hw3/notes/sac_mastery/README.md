# SAC Mastery Notes

This folder implements the 1-week SAC mastery plan with concrete study artifacts tied to the current `hw3` codebase.

Files:
- `day1_mental_model.md`: actor / critic / target critic / temperature overview
- `day2_critic_update.md`: critic target formula, data flow, and tensor shapes
- `day3_actor_update.md`: reparameterization, actor loss, and common sign mistakes
- `day4_temperature_and_entropy.md`: fixed temperature vs auto-tune
- `day5_experiment_log.md`: sanity and HalfCheetah experiment observations
- `day6_sac_vs_dqn.md`: comparison against the current DQN implementation
- `day7_teachback.md`: short teaching summary and debugging checklist

Suggested order:
1. Read `day1_mental_model.md`
2. Work through `day2_critic_update.md`
3. Work through `day3_actor_update.md`
4. Finish `day4_temperature_and_entropy.md`
5. Check `day5_experiment_log.md` while looking at real logs
6. Use `day6_sac_vs_dqn.md` to strengthen intuition
7. Review `day7_teachback.md` until you can explain SAC without notes
